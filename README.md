## ğŸš€ Getting Started

### Clone the Repository

```bash
git clone https://github.com/masoomjethwa/masoomjethwa.github.io.git
cd masoomjethwa.github.io
```

### Local Development

1. **Start a local server** (Python):
```bash
# Python 3
python -m http.server 8000

# Python 2
python -m SimpleHTTPServer 8000
```

2. **Open in browser**:
   - Navigate to `http://localhost:8000`
   - Visit `http://localhost:8000/index.html`

### Customize for Your Use

1. **Edit main index.html** - Update personal information
2. **Modify style.css** - Change colors and styling
3. **Update script.js** - Customize GitHub username
4. **Add content** - Create new folders and pages
5. **Commit & Push** - Deploy to GitHub Pages
